what it is

A google chrome extension that allows you to open multiple links in one quick go. Use your mouse to select all the links you want and they will open up in new tabs.

how to use

Windows: by default you press-and-hold the right mouse button and drag a box over the links you want to open.

Mac/Linux: by default you press-and-hold the z key and the left mouse button and drag a box over the links you want to open.

features

Customise: you can change how Linkclump is activated from the popdown menu. Please note that a key is mandatory when using the right mouse button on Mac/Linux operating system.

Smart select: with this option enabled only the relevant links are selected. This feature is especially useful on search result pages.

Exclude words: you can exclude links that contain certain words.

feedback & bugs
